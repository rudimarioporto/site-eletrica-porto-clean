# Elétrica Porto

## 1. Descrição do Projeto
Site institucional para a **Elétrica Porto**, empresa especializada em manutenção de motores elétricos e bombas d'água. Objetivo: apresentar a empresa, gerar confiança e permitir que clientes solicitem serviços/agendamentos com facilidade. Público-alvo: residencial, comercial e rural.

## 2. Estrutura de Páginas
- `/` - Página única (one-page) com âncoras:
  - `#inicio` - Hero
  - `#sobre` - Sobre a Empresa
  - `#servicos` - Serviços
  - `#agendamento` - Agendamento
  - `#depoimentos` - Depoimentos
  - `#dicas` - Dicas Rápidas
  - `#contato` - Contato / Footer

## 3. Funcionalidades Principais
- [x] Navbar responsiva com hambúrguer mobile
- [x] Hero com CTA (Orçamento / Agendamento)
- [x] Seção Sobre com valores da empresa
- [x] Seção Serviços (4 categorias)
- [x] Formulário de Agendamento com WhatsApp automático
- [x] Confirmação visual após envio
- [x] Formulário de Contato
- [x] Depoimentos de clientes
- [x] Seção Dicas Rápidas
- [x] Botão flutuante WhatsApp
- [x] Botão Salvar Contato (.vcf)
- [x] Footer completo

## 4. Integrações
- Supabase: Não necessário
- Shopify: Não necessário
- Stripe: Não necessário
- Formulários: Readdy Forms (agendamento + contato)
- WhatsApp: Link direto com mensagem pré-preenchida
